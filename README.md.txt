# Milk FM Alexa Skill

Este repositorio contiene el código para la skill de Alexa de la emisora **Milk FM**.

## Instrucciones
1. Clona este repositorio.
2. Configura la skill en la [Alexa Developer Console](https://developer.amazon.com/alexa/console/ask).
3. Despliega el código en AWS Lambda o Alexa-Hosted.

## Comandos de voz
- "Abre Milk FM" o "Reproduce Milk FM": Inicia la reproducción.
- "Pausa": Pausa la reproducción.
- "Reanuda": Reanuda la reproducción.
- "Detén": Detiene la reproducción.
